Oriz MF NAV API

Daily Indian mutual fund NAV snapshots — clean JSON, PWA-installable docs.

Data source: This API proxies the free, open api.mfapi.in (originally from AMFI India). All credit for collection and curation goes to them. We add cleaner shape + daily snapshot archival.

Endpoints

  • GET /data/latest.json — most recent snapshot
  • GET /data/mf-nav-YYYY-MM-DD.json — historical daily snapshot
